title: "AI 动作"

在自动化中，动作是触发器激活后要执行的操作之一。
Teable AI动作通过提供自然语言指令（Prompt）与结构化数据（字段数值）的智能结合，允许用户实现对表格数据的动态解析与内容生成。

### 配置过程
- 输入提示 
  - 输入自定提示词。
  - 点击输入框右上角的“+”号选择表格中的数据。可以使用其他节点生成的数据作为数据来源，也可以从表格中的字段插入值。

![](/images/2025-04-11-12-14-31.png)

- 选择模型
  - 选择模型的时候，可以根据任务需求选择不同类型的模型，每种模型具备独特的处理能力：
  - 视觉模型 
    - 支持附件识别（如提取图中文字、分析图片内容）
    - 兼容常见文件格式：Word/Excel/PDF（自动解析其中嵌入的文字、图片或图表）
  - 推理模型 
    - 提供完整推理链，展示分析过程的每一步逻辑
    - 适用于需要透明决策依据的任务（如数据验证、复杂问题拆解）

![](/images/2025-04-17-11-50-40.png)

- 选择温度
  - 温度(Temperature)是一个用于控制人工智能生成文本的创造力水平的参数。 通过调整“温度”，您可以影响AI模型的概率分布，使文本更加确定或更多样化。
  - 低温态（0.2-0.5）：输出确定的结果（如内容整理、数据分析）
  - 中温态（0.6-0.8）：平衡准确性与可读性（如报告撰写、回复邮件）
  - 高温态（0.9-1.2）：输出更具创造性的表达（如文案构思、故事创作）
- 选择输出类型
  - 支持文本和 JSON 两种格式。
  - JSON 格式的输出便于数据的进一步分析和处理，例如从发票图片中识别出的关键信息（金额、日期等），方便后续自动化流程调用。

### 测试
  - 配置完成后，点击“测试”按钮。
  - 系统会显示动作的测试结果，可以在此查看输出结果并调整提示。

![](/images/2025-04-17-11-38-05.png)

### Used Credits
  - 在测试的输出栏目可以查看Credits的使用量。
  - 在 Cloud 版本中，usedCredits 代表本次调用模型使用的 credits。使用量根据输入和输出的 token 计算得来。
不同模型的 token 与 credit 的换算比例可以在模型选择时候查看。


<Tip> 除了Teable官方提供的模型，Teable还允许使用第三方模型 API，可以在空间→设置中进行设置。 </Tip>
