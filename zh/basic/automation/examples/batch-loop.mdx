title: "用循环批量处理记录"
description: "使用查询记录 + 循环在一次工作流运行中批量处理多条记录"
---

**目标：** 每天查找所有逾期任务，并将其状态标记为"逾期"。

## 步骤

1. **创建工作流**，触发器选择 **定时执行** —— 每天 08:00。
2. **添加操作** → **查询记录**。
   - 表格：**任务**
   - 筛选条件：`截止日期` **早于** `今天` AND `状态` **不是** `逾期`
3. 点击**测试**查看匹配的记录。
4. **添加操作** → **更新记录**。
5. 点击表格选择器下方的**循环执行**。
6. 选择查询记录步骤输出的数组。
7. 映射：
   - 记录 ID → 当前循环项的记录 ID（通过 **+** 插入）
   - 状态 → `逾期`
8. 点击**测试**，然后**启用**。

## 工作原理

查询记录步骤返回一个数组。启用循环执行的更新记录步骤会遍历数组中的每一项，按顺序逐条更新。
