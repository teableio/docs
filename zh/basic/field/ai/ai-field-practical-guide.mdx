# AI 字段工作流：从 0 到 1 打造自动化营销实战

<Tip> 您是否曾想过，从一个产品概念到生成多平台营销文案、创作视觉素材、完成多语言适配，再到分析市场反馈，这一切都可以在 Teable 完成？现在，Teable 的 AI 字段组合拳，将这个设想变为现实。 </Tip>

---

## 从 0 到 1 打造“智能降噪咖啡杯”营销战役

### 第 0 步：播下种子 - 基础信息

我们从一个简单的表格开始，只需填入两个基础字段：

-   `[产品名称]`: 智能降噪咖啡杯
-   `[核心卖点]`: 主动降噪技术，12小时超长续航，食品级316不锈钢

![基础信息表格](/images/2025-07-21-15-43-14.png)

### 第 1 步：引爆内容 - AI 文本创作

我们新建 3 个 AI 字段，让它们引用基础信息，一键生成适配不同平台的内容：

1.  **`[AI] 营销长文案`**:
    ```
    为产品{产品名称}写一篇详细的营销文案，突出其{核心卖点}，风格要科技感十足。
    ```

2.  **`[AI] 小红书笔记`**:
    ```
    模拟一个数码博主的口吻，为{产品名称}写一篇小红书种草笔记，包含吸引人的标题和emoji。
    ```

3.  **`[AI] 朋友圈广告`**:
    ```
    用不超过50字的精炼文案，为{产品名称}写一则朋友圈广告。
    ```

### 第 2 步：眼见为实 - AI 图像生成

<Note>
生成图片（附件字段）前请先配置文生图模型。操作路径：空间管理 > 集成 > 添加自定义AI模型。
</Note>

内容有了，配图怎么办？我们再添加一个 AI （附件类型）字段，让它引用刚刚生成的内容来创作图像：

1.  **`[AI] 产品概念图`**:
    ```
    根据以下描述，生成一张产品在现代办公室场景下的高清渲染图，风格简约高级：{AI 营销长文案}
    ```

### 第 3 步：走向世界 - AI 翻译与本地化

准备进军海外市场？轻松搞定。

1.  **`[AI] 英文营销文案`**:
    ```
    将{AI 营销长文案}翻译成地道的、适合北美市场的英文。
    ```

2.  **`[AI] 英文宣传图`**:
    ```
    Generate a photorealistic image for an English ad campaign based on this description: {AI 英文营销文案}
    ```

### 第 4 步：倾听回响 - AI 反馈分析

产品发布后，将收集到的用户评论粘贴到另一个表格中。AI 字段将自动进行深度分析：

1.  **`[AI] 情感判断`**:
    ```
    分析{用户评论原文}的情感，只回答‘正面’、‘负面’或‘中性’。
    ```

2.  **`[AI] 提取关键词`**:
    ```
    从{用户评论原文}中提取3个核心关键词。
    ```

3.  **`[AI] 问题归类`**:
    ```
    将{用户评论原文}的问题归类为：功能建议、外观设计、电池续航、价格咨询。
    ```

### 第 5 步：探索性数据分析

现在，我们不再需要手动筛选和透视分析，而是直接与数据对话。

我们打开表格的 **AI 对话** 功能。它就像一位7x24小时在线的数据分析师。

-   **向AI提问**：“我们收到的负面反馈主要集中在哪些方面？请列出前三名。”
-   **AI 回答**：“根据数据分析，负面反馈主要集中在：1. 降噪效果（尤其对人声）；2. 电池续航；3. 杯身材质易留指纹。”

无需复杂的公式和筛选器，通过自然语言对话，我们就快速洞察了用户反馈的核心。

![AI对话功能分析数据动图](/images/2025-06-16-19-56.gif)

### 第 6 步：自动化数据可视化

对话得出了结论，现在需要向团队汇报。

-   **向AI下令**：“根据刚才的分析，生成一张按问题类型分类的用户反馈数量的饼图。”

瞬间，一张清晰、专业的饼图就出现在您眼前。

![AI对话功能生成图表动图](/images/2025-06-16-22-20.gif)

从数据到图表，从分析到报告，整个过程在几次对话中无缝完成。

## 最终闭环：从洞察到行动的零距离

通过这套AI组合拳，我们不仅知道了用户抱怨“降噪效果”，更通过对话深挖出是“对高频人声”的降噪问题。我们不仅知道了用户喜欢什么，更能一键生成图表，在团队会议上清晰地展示出来。

最终，决策变得无比简单和精准：

1.  **产品团队**：收到包含具体问题分类饼图的报告，立即着手优化下一代产品针对人声的降噪算法。
2.  **营销团队**：获得用户最爱卖点的条形图，调整宣传策略，主推“设计感”与“恒温体验”。
