每个数据库可以包含多个表格，用于收集、存储和处理相关数据。Teable 的表格等同于关系数据库中的表，与 Excel 工作表有所不同——了解[两者的区别](/zh/compare/teable-vs-excel)有助于更好地掌握 Teable。

### 新建表格

在数据库中点击左侧目录顶部的 `+` 按钮，可选择：

<img
  src="/images/docs/base/2026-05-22-table-create-menu.png"
  alt="新建表格菜单"
  className="docs-screenshot"
/>

1. 新建空白表格
2. 从 CSV 导入
3. 从 Excel 导入

具体操作参考 [导入](/zh/basic/table/import) 章节。

### 重命名与删除

<img
  src="/images/docs/base/2026-05-22-table-rename-delete.png"
  alt="重命名与删除表格"
  className="docs-screenshot"
/>

- **重命名**：双击目录中的表格名称，输入新名称后点击空白处自动保存
- **删除**：鼠标悬停在表格名称上，点击 `...` 菜单，选择 **删除**


### 回收站

点击数据库界面左上角的 **回收站** 图标，可对已删除的表格执行：

<img
  src="/images/docs/base/2026-05-22-table-trash.png"
  alt="表格回收站"
  className="docs-screenshot"
/>

- **恢复**：将表格还原到数据库
- **清空**：永久删除，无法恢复

### 设计

点击表格菜单中的 **设计**，可查看表格基础属性：

<img
  src="/images/docs/base/2026-05-22-table-design.png"
  alt="表格设计"
  className="docs-screenshot"
/>

**表格信息**

所属 Schema、物理表名、描述、最后修改时间

**字段明细**

| 属性名 | 描述 |
| :--- | :--- |
| **id** | 字段 ID |
| **name** | 字段名称 |
| **dbFieldName** | 字段在物理数据库中的名称 |
| **type** | 字段类型 |
| **description** | 字段描述 |
| **graph** | 查看当前字段依赖关系 |
| **cellValueType** | 当前字段值类型 |
| **isLookup** | 是否为从关联表查找出来的字段 |
| **isMultipleCellValue** | 是否为[数组值字段](/zh/basic/field/common/is-multiple-value) |
| **isComputed** | 是否为计算字段（计算字段的记录值不可修改） |
| **isPending** | 是否为计算中 |
| **hasError** | 是否存在计算错误 |
| **notNull** | 是否启用非空校验 |
| **unique** | 是否启用唯一值校验 |

### 分享表格

在表格中点击右上角 **分享**，切换到 **分享表**，打开 **公开分享到网络**，即可生成分享链接。

<img
  src="/images/docs/base/2026-05-22-table-share-entry.png"
  alt="表格分享入口"
  className="docs-screenshot"
/>
<img
  src="/images/docs/base/2026-05-22-table-share-web.png"
  alt="公开分享表格"
  className="docs-screenshot"
/>

可配置链接访问权限：

<img
  src="/images/docs/base/2026-05-22-table-share-permissions.png"
  alt="表格分享权限"
  className="docs-screenshot"
/>

- **可查看**：获得链接的人可以查看表格数据
- **可编辑**：登录用户可以编辑分享表格中的记录
- **可另存为副本**：获得链接的人可以把分享内容保存到自己的空间

还可以配置 **允许查看者复制数据**、**需要密码访问**、重新生成或删除链接和嵌入配置。

如需分享当前视图，请切换到 **分享视图**，详见 [分享视图](/zh/basic/view/toolbar/share)。

### 访问表格

新建表格默认包含[表格视图](/zh/basic/view/grid)，也可通过 [API](/zh/api-doc/overview) 访问。用户可创建[看板](/zh/basic/view/kanban)等多种视图以满足不同场景需求。

### 全局搜索

<img
  src="/images/docs/base/2026-05-22-table-global-search.png"
  alt="表格全局搜索"
  className="docs-screenshot"
/>

- 除文本字段外，**日期字段也可以参与全局搜索**，直接搜索日期值即可定位对应记录
- 选择隐藏不匹配的行时，结果范围会随搜索条件一起收窄

### 附件下载

从附件字段菜单批量下载时，仅下载当前搜索结果中的附件。

<img
  src="/images/docs/base/2026-05-22-table-attachment-download-menu.png"
  alt="附件批量下载入口"
  className="docs-screenshot"
/>

在批量下载弹窗中，可通过 **附件名前缀** 选择 ZIP 内文件的命名方式：默认序号、指定字段作为前缀，或 **无前缀** 保留原始文件名；重名文件会自动追加后缀。

<img
  src="/images/docs/base/2026-05-22-table-attachment-download-options.png"
  alt="附件批量下载选项"

### 常见问题

<AccordionGroup>
  <Accordion title="为什么数字列不能填文字？">
    Teable 比 Excel 更严格——每列都有固定的数据类型，数字列只能存储数字。这种约束确保了数据规范，是实现高级筛选和自动化的基础。更多细节请看 [Teable vs Excel](/zh/compare/teable-vs-excel)。
  </Accordion>
  <Accordion title="删除表格前需要注意什么？">
    请检查该表格是否通过**关联字段**被其他表格引用。如果删除了源头表格，可能会导致关联表格中的数据显示异常。
  </Accordion>
</AccordionGroup>
