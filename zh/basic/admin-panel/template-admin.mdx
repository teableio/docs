### 模板管理

在模板管理界面，您可以创建和配置将显示在组织模板中心的模板。

#### 模板信息

* **封面**：模板卡片显示的封面图片
* **名称和描述**：在模板中心显示的标题和简短介绍
* **Markdown 描述**：支持富文本格式的详细介绍
* **分类**：按业务场景对模板进行分组（如：人力资源、销售、研发等）
* **系统模板**：标识是否为官方系统模板
* **来源**：创建此模板所使用的原始数据库

#### 模板操作

* **新建模板**：创建新的模板配置
* **发布快照**：捕获源数据库的当前状态。必须先发布快照才能上架模板。
* **状态 (上架)**：控制模板是否在模板中心可见。上架前需要填写名称、描述并发布快照。
* **置顶**：将重要模板固定在列表顶部
* **删除**：移除模板配置

### 使用官方模板

私有化部署用户可以导入 Teable 官方云端的模板：

1. **下载**：从 [官方模板中心](https://teable.cn/templates) 下载 `.tea` 文件
2. **导入**：将 `.tea` 文件作为新数据库导入到您的空间中
3. **新建模板**：进入 管理后台 > 模板管理 > 新建模板
4. **选择来源**：在来源字段选择刚才导入的数据库
5. **上架**：配置详情，发布快照，并开启上架状态
