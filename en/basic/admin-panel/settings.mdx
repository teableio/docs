<Tip>Available for self-hosted Pro plan and above</Tip>

### General Settings

* **Allow creating new accounts**: Enable to show a registration button on the login page.
* **Allow sending space invitations**: Enable to allow users to invite others to spaces.
* **Allow everyone to create new spaces**: Enable to allow all users to create spaces.
* **Enable email verification**: Enable to require email verification for new accounts.

### Email Config

Configure your SMTP server for system emails.

* **Notification Email**: Used for system notifications (e.g. password reset, welcome emails).
* **Automation Email**: Used for sending emails via automation actions.

### Pending configuration

This card on the right side highlights important system-level configurations that should be completed to ensure all features work properly.

![](/images/pending-configuration.png)

* **Enable https**: Required for large copy-paste operations.
* **PUBLIC_DATABASE_PROXY environment variable**: Required for external database connections.
* **LLM API**: Required for AI features (Chat, Automation).
* **App Generation**: Required for AI app generation.
* **Web Search**: Required for AI web search capabilities.
* **Email**: Required for password recovery and notifications.
