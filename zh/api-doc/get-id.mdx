title: "获取 ID"

## 空间 ID (SpaceId)

点击目标空间，复制 URL 上以 spc 开头的字符串，示例: `spcXXXXXXXXXX`

![](/images/kai-fa-zhe/api/fa9fb4d225e32565.png)

![](/images/kai-fa-zhe/api/b7c3b29073bc6768.png)

## 数据库 ID (BaseId)

点击目标数据库，复制 URL 上以 bse 开头的字符串，示例：`bseXXXXXXXXXX`

![](/images/kai-fa-zhe/api/9f35f5ecaf38f264.png)

![](/images/kai-fa-zhe/api/17f341d5b0b07ff4.png)

## 表格 ID (TableId)

点击目标表格，复制 URL 上以 tbl 开头的字符串, 示例：`tblXXXXXXXXXX`

![](/images/kai-fa-zhe/api/88aefa10b5ffe372.png)

![](/images/kai-fa-zhe/api/99efd16eae04feac.png)

## 视图 ID (ViewId)

点击目标视图，复制 URL 上以 viw 头的字符串, 示例：`viwXXXXXXXXXX`

![](/images/kai-fa-zhe/api/830177806519ba17.png)

![](/images/kai-fa-zhe/api/d7139d0c9c0960b5.png)

## 字段 ID (FieldId)

**方法 1：**

点击表格左上角 `更多` 中的 `设计` 按钮，进入设计界面，查看所有字段的 ID

<img src="/images/2026-02-12-10-50-30.png" width="400" />

![](/images/2025-02-14-18-16-26.png)

**方法 2：**

双击列标题或选择“编辑字段”后打开`高级属性`。在`物理字段名`部分，您可以找到字段 ID。

<img src="/images/2026-02-12-10-59-21.png" width="400" />

<img src="/images/2026-02-12-11-00-16.png" width="400" />

## 记录 ID (RecordId)

展开记录编辑卡片，复制 URL 上 recordId= 后方以 rec 开头的字符串, 示例：`recXXXXXXXXXX`

![](/images/kai-fa-zhe/api/3a3abbbb21e81c7c.png)
