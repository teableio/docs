数字字段不仅仅是简单的数字记录，还具备以下特性：
- **自动校验**：仅允许输入整数或小数（如"12"、"3.14"），自动拦截非数字文本
- **格式多样**：支持常规数字、货币符号、百分比三种格式化显示
- **可视化展示**：支持将数值转化为条形或环形，直观呈现数据状态

## 一、 创建与配置

### 创建数字字段

- **步骤**：
    1. 点击表格视图表头右侧的 `+`，在字段类型列表中选择 **数字**
    2. 输入字段名称（如"商品价格"、"库存数量"）
    3. **格式设置**：根据业务需求选择数据的显示类型
        - **数字**：纯数值显示，适用于普通计数（如：100、2.5）
        - **货币**：自动添加货币符号，适用于金额统计（如：¥100.00）
        - **百分比**：以百分比形式显示，适用于比率统计（如：50%）

<img alt="设置格式化" style={{ maxWidth: '65%' }} src="/images/2025-11-26-17-06-08.png" className="mx-auto" />

    4. **精度设置**：设置小数点后的保留位数
        - 提供从"整数"到"保留4位小数"共5个等级供选择
        - **注**：设定精度后，系统会自动对输入数据进行四舍五入显示

<img alt="设置精度" style={{ maxWidth: '65%' }} src="/images/2025-11-26-17-10-28.png" className="mx-auto" />

### 显示设置
除了基础数值，还可以通过图形化方式增强数据的可读性。

- **步骤**：
    1. 在字段配置页面的"显示方式"栏中进行选择
    2. **默认**：仅显示纯文本数值
    3. **环形 / 条形**：将数值转化为可视化图形
        - **目标值**：设置图形满格对应的数值（如满分为100，则填100）
        - **颜色**：自定义图形的填充颜色
        - **显示数字**：勾选后可在图形旁同时显示具体数值

## 二、 使用场景

### 场景一：财务管理 - 合同金额记录
- **数据表前提**：合同管理表
- **需求场景**：需记录每份合同的金额，要求精确到分，并显示人民币符号
- **操作步骤**：
    1. 新建字段命名为"合同金额"
    2. 格式选择 **货币**，符号选择 **¥**
    3. 精度选择 **保留2位小数**（如：¥1,000.00）

### 场景二：目标管理 - 销售业绩达成率
- **数据表前提**：销售业绩表
- **需求场景**：直观展示销售人员的业绩完成度，以便快速识别是否达标
- **操作步骤**：
    1. 新建字段命名为"达成率"
    2. 格式选择 **百分比**
    3. 显示方式选择 **条形**
    4. 设置目标值为 **1**（即100%），选择绿色作为进度颜色

## 三、 功能特性

数字字段具备以下核心功能：
- **自动校验**：仅允许输入整数或小数（如"12"、"3.14"），自动拦截非数字文本
- **格式多样**：支持常规数字、货币符号、百分比三种格式化显示
- **可视化展示**：支持将数值转化为条形或环形，直观呈现数据状态

## 四、 注意事项

- 百分比采用按小数存储的机制，因此输入 "100%" 实际存储为 "1"，输入 "50%" 实际存储为 "0.5"
