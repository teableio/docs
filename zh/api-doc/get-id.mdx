title: "获取 ID"
---

## 空间 ID (SpaceId)

点击目标空间，复制 URL 上以 spc 开头的字符串，示例: `spcXXXXXXXXXX`

<img
  src="/images/api/get-id/2026-05-29-get-id-space-id.png"
  alt="URL 中的空间 ID"
  className="docs-screenshot"
/>

## 数据库 ID (BaseId)

点击目标数据库，复制 URL 上以 bse 开头的字符串，示例：`bseXXXXXXXXXX`

<img
  src="/images/api/get-id/2026-05-29-get-id-base-id.png"
  alt="URL 中的数据库 ID"
  className="docs-screenshot"
/>

## 表格 ID (TableId)

点击目标表格，复制 URL 上以 tbl 开头的字符串, 示例：`tblXXXXXXXXXX`

<img
  src="/images/api/get-id/2026-05-29-get-id-table-id.png"
  alt="URL 中的表格 ID"
  className="docs-screenshot"
/>

## 视图 ID (ViewId)

点击目标视图，复制 URL 上以 viw 头的字符串, 示例：`viwXXXXXXXXXX`

<img
  src="/images/api/get-id/2026-05-29-get-id-view-id.png"
  alt="URL 中的视图 ID"
  className="docs-screenshot"
/>

## 字段 ID (FieldId)

**方法 1：**

点击表格左上角 `更多` 中的 `设计` 按钮，进入设计界面，查看所有字段的 ID

<img
  src="/images/api/get-id/2026-05-29-get-id-field-id-design-entry-zh.png"
  alt="从更多菜单进入设计页面"
  className="docs-screenshot"
/>

<img
  src="/images/api/get-id/2026-05-29-get-id-field-id-design-list.png"
  alt="设计页面中的字段 ID"
  className="docs-screenshot"
/>

**方法 2：**

双击列标题或选择“编辑字段”后打开`高级属性`。在`物理字段名`部分，您可以找到字段 ID。

<img
  src="/images/api/get-id/2026-05-29-get-id-field-id-edit-field-entry-zh.png"
  alt="从字段菜单打开编辑字段"
  className="docs-screenshot"
/>

<img
  src="/images/api/get-id/2026-05-29-get-id-field-id-db-field-name-zh.png"
  alt="物理字段名中的字段 ID"
  className="docs-screenshot"
/>

## 记录 ID (RecordId)

展开记录编辑卡片，复制 URL 上 recordId= 后方以 rec 开头的字符串, 示例：`recXXXXXXXXXX`

<img
  src="/images/api/get-id/2026-05-29-get-id-record-id.png"
  alt="记录卡片 URL 中的记录 ID"
  className="docs-screenshot"
/>
