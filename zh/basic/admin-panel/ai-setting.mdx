#### 1. AI 能力设置
- AI 基础能力：包括 AI 表格查询工具、执行 SQL 查询工具、生成脚本工具、表格数据可视化工具。
- 创建数据库：从 0 开始创建整个数据库。
- 创建应用：基于数据库创建应用。
- 创建自动化：从 0 开始创建整个自动化。
- 表格资源操作：通过 AI 对表格资源操作，可以创建、修改、删除表格、视图、字段、记录。
- 建议：根据当前数据库数据以及对话内容，给予建议。

#### 2. 自定义模型
- 启用AI 能力将使用您的自定义模型并消耗您自己的 Token。

#### 3. 模型提供商与模型偏好

##### 步骤一：添加模型提供商

1.  点击 `+ 添加模型提供商` 按钮。
2.  在弹出的表单中，选择您的模型服务商。
3.  输入该服务商提供的 API 密钥（API Key）和其他必要信息并保存。

提示：在您添加并正确配置模型提供商之前，系统会提示“您尚未配置 AI LLM API”，所有依赖大语言模型的功能（如 AI 对话、AI 自动化等）都将无法使用。

##### 步骤二：设置模型偏好

添加提供商后，您需要为不同复杂度的任务指定偏好的模型，以平衡成本、速度和能力。

*   对话模型
    *   高级对话模型: 用于处理最复杂的任务，如详细的规划、代码生成、深度分析等。推荐使用能力最强的模型（如 Claude 3 Sonnet, GPT-4）。
    *   中型对话模型: 用于平衡速度和质量的日常任务，如创建数据库、常规对话等。推荐使用性能均衡的模型（如 GPT-4o）。
    *   小型对话模型: 用于快速、简单的任务，如场景推荐、快速操作等。推荐使用速度快、成本低的模型。

![](/images/2025-10-10-21-04-33.png)

*   高级对话模型能力
    *   图片 (Image) / PDF: 启用这些选项可以让您的高级模型具备视觉能力，能够理解和分析您上传的图片或 PDF 文档内容。
    *   测试模型: 点击此按钮可以验证您选择的高级模型及其 API 密钥是否配置正确且可用。

*   嵌入模型
    *   作用: 此模型用于将文本转换为向量。
    *   配置: 通常选择模型 ID 中带有 "embedding" 字样的模型。

![](/images/2025-10-10-21-04-44.png)

#### 4. 其他 AI 功能集成

此区域用于配置特定的、需要独立 API 密钥的 AI 功能。

*   应用生成
    *   功能: 允许 AI 从一个简单的文本提示生成一个功能完整的应用程序。
    *   配置: 需要一个 `v0` API 密钥。请访问 v0 的官方设置页面获取您的密钥，并在此处输入。如果未配置，系统会提示“您尚未配置 v0 API”。

*   网络搜索
    *   功能: 赋予 AI 实时访问互联网、搜索最新信息的能力。
    *   配置: 本功能通过集成 `Firecrawl` 服务实现。请访问 Firecrawl 的设置页面获取您的 API 密钥，并在此处输入。如果未配置，系统会提示“您尚未配置网络搜索 API”。
