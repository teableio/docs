### 模板管理入口
进入应用，从左侧边栏管理面板进入管理后台，选择模板管理界面。

![](/images/image.png.png)

### 设置
在模板管理界面可以对模板进行配置。
点击新建模板后，会显示模板上架所需要的所有信息，包括：
- 添加封面
- 设置名称和描述
- 选择分类：支持直接选择和搜索选择分类，并允许自主添加分类。
- 系统模板：展示哪些模板来源于Teable 官方（后续支持）
- 发布快照：快照是模板在某个时间点上的状态。点击发布快照后，将上传模板在发布快照时的状态。快照功能允许用户在上架模板后对原数据库中的模板进行修改，减少因模板版本差异引发的分歧。
- 快照时间：点击发布快照时的具体时刻。
- 上架：
  - 选择是否上架之前，需要先发布快照，并确保名称和描述不为空。
  - 上架后，其他用户才可以在创建数据库的时候使用该模版
- 来源：选择模板来源的数据库
- 操作：可以将模板在模板管理和模板中心中置顶或者删除



### 模板管理

- 自动化内容管理
    - 如果模板中心提供的模板已经配置好自动化，在表格内按照表格场景更新数据，支持根据表格现有字段类型进行生成。
- 数据展示
    - 如果模板中心提供的模板已经配置好仪表盘，表格内的所有数据支持直接通过仪表盘展示。
- 与团队成员共享模板 


### 使用 teable.cn 的模板
私有化部署的用户不仅可以在组织内创建模板，还可以导入来自其他 Teable 环境（如官方）的模板，并将其上架到自己的模板中心，方便组织内成员使用。

操作流程如下：

1.  **获取模板 .tea 文件**
 <Note>[官方模版中心](https://app.teable.cn/share/shrw5mmsCrKCQnRQdkk/view)</Note>
    - 找到并下载您需要的模板（如：“产品周报模板”）。

2.  **导入为数据库**：在您的环境中，将下载的 `.tea` 文件导入，它会成为您空间中的一个新数据库。

3.  **进入模板管理**：使用管理员账号，进入「**管理后台** > **模板管理**」界面。

4.  **新建并配置模板**：点击**新建模板**，然后在**来源**字段中，选择您刚刚通过文件导入的那个数据库。

5.  **发布与上架**：填写模板的名称、描述、分类等信息，点击**发布快照**，最后点击**上架**开关。上架成功后，该模板就会出现在您组织的模板中心，供所有成员使用。
