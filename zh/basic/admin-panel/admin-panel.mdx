### 管理后台入口

管理员进入应用，点击左上角管理面板进入管理后台

![](/images/ji-ben-gong-neng/a248626c931471b3.png)

### 设置

在设置界面可以配置通用设置，包括：

* 允许新用户注册：关闭此选项将禁止新用户注册，登录页面将不再显示注册按钮。

* 允许发送空间邀请： 关闭此选项将禁止除管理员以外的用户邀请他人加入空间。开启此选项后，即使新账户注册已被禁用，仍允许被直接邀请的用户创建账户。

* 允许所有人创建新的空间： 关闭此选项将禁止除管理员以外的用户创建新的空间。

以及当前实例 ID 和应用版本

![](/images/ji-ben-gong-neng/30b53d0786df60f0.png)

**许可证**

点击注册许可证按钮，可以填写[购买的许可证](https://app.teable.cn/setting/license-plan)，激活应用。

![](/images/ji-ben-gong-neng/cb23e28c58fe2991.png)

注册后该页面会显示许可证类别，席位数量，有效期等信息，点击更新许可证按钮可以进行续期

![](/images/ji-ben-gong-neng/3e0dc36fb6c5f104.png)

#### 配置项

除了上述相关配置外，系统右侧的“待配置项”卡片中可能还会提示您完成以下系统级配置，以确保功能都能正常运行。

-   启用 HTTPS:
    -   说明: 您的实例尚未启用 HTTPS 安全连接。
    -   影响: 缺少 HTTPS 会影响系统安全，并且某些依赖安全连接的功能（如大规模数据复制）将无法使用。

-   PUBLIC_DATABASE_PROXY 环境变量:
    -   说明: 用于连接外部数据库的代理服务未配置。
    -   影响: 您将无法使用平台连接和操作外部的数据库资源。

-   邮箱:
    -   说明: 系统邮件服务未配置。
    -   影响: 用户将无法使用自助密码恢复功能，也无法接收系统发送的邮件通知。

![](/images/2025-10-10-20-18-29.png)

### 空间管理

点击左侧空间管理按钮可以进入空间管理界面，这里会显示当前实例下的所有空间信息。包括：

* 空间名称
* 数据库总数量
* 协作者总数量
* 创建时间
* 是否自动加入

并且可以在该空间执行操作

* 删除：删除空间后空间会进入回收站
* 允许自动加入：开启后，所有新注册的用户会自动进入该空间，并赋予[可阅读权限](/zh/basic/space/space-invite)

![](/images/ji-ben-gong-neng/fe637cdf26bbd5e1.png)

### 用户管理

点击左侧用户管理按钮可以进入用户管理界面，这里会显示当前实例下的所用户信息，包括

* 用户名
* 邮箱
* 最后登录时间
* 注册时间
* 是否激活状态

并可以进行操作

* 停用：该用户无法登陆，但保留用户信息，可以被查找
* 删除：删除该用户信息

![](/images/ji-ben-gong-neng/2957241a218c70d8.png)
