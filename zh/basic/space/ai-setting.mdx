### 1. AI 能力设置

- 说明：
  - 空间管理员可以独立控制每一项 AI 功能是否在空间内可用。
  - 列表中的功能包括：
    - AI 基础能力：包括 AI 表格查询工具、执行 SQL 查询工具、生成脚本工具、表格数据可视化工具。
    - 创建数据库：从 0 开始创建整个数据库。
    - 创建应用：基于数据库创建应用。
    - 创建自动化：从 0 开始创建整个自动化。
    - 表格资源操作：通过 AI 对表格资源操作，可以创建、修改、删除表格、视图、字段、记录。
    - 建议：根据当前数据库数据以及对话内容，给予建议。

![](/images/2025-09-25-21-36-58.png)

### 2. 自定义 AI 模型

- 用于激活或停用当前空间内的自定义 AI API 配置。启用该配置后，将会优先使用空间 AI 配置。
- 说明：
  - 关闭时：使用官方提供的默认 AI 模型。
  - 打开时：将使用自定义配置的 AI 模型（消耗您自己的 Token），开启前，请配置好高级对话模型。


> [自定义模型配置指南](/zh/basic/ai/custom-model)

### 3. 模型提供商

- 用于管理自定义配置 AI 模型提供商的接入。



![](/images/2025-10-10-19-36-09.png)

### 4. 模型偏好

- 为不同类型的任务场景指定默认使用的对话模型，优化成本和性能。
  - 高级对话模型：
    - 用于 AI Chat 主对话、复杂推理和代码生成任务。
    - 推荐：`claude-sonnet-4.5` 或具有强推理能力的模型。
  - 中型对话模型：
    - 用于 AI 公式生成。
    - 未配置时会回退到高级对话模型。
    - 推荐：`claude-haiku-4.5`
  - 小型对话模型：
    - 用于自动生成对话标题和数据库名称等轻量级任务。
    - 未配置时会回退到高级对话模型。
    - 推荐：`claude-haiku-4.5`

### 5. 高级对话模型能力

- 测试所选"高级对话模型"的多模态处理能力。
- 说明：
  - 可以通过"测试模型"按钮来验证所选的高级模型是否支持处理图片和PDF内容。

### 6. 嵌入模型

- 可选。用于文档问答 - 当用户上传文档并询问文档内容相关问题时使用。通常模型 ID 包含 "embedding"。

![](/images/2025-09-25-21-36-33.png)

### 7. 应用构建器 (App Builder)

配置 v0 API 密钥以启用 [应用构建器](/zh/basic/ai/app-builder) 功能。访问 [v0.app/chat/settings/keys](https://v0.app/chat/settings/keys) 获取 API 密钥。

### 8. 网络搜索

- 配置 Firecrawl API 密钥以启用网络搜索功能，访问 Firecrawl 设置 获取 API 密钥

![](/images/2025-10-10-19-38-21.png)
