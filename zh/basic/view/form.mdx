---
title: "表单"
description: "用表单视图收集外部或内部提交，并把每次提交保存为当前表中的新记录。"
---

表单视图会基于当前表格生成表单。每次提交都会在创建表单视图的表中新增一条记录，适合收集反馈、报名信息、报销申请、客户资料和其他标准化数据。

## 适用场景

| 场景 | 适合内容 |
| --- | --- |
| 信息收集 | 客户资料、报名表、问卷反馈 |
| 内部申请 | 报销、采购、请假、权限申请 |
| 文件提交 | 简历、凭证、合同附件、截图 |
| 轻量流程入口 | 将外部填写内容直接沉淀到表格记录中 |

## 创建表单视图

<Steps>
  <Step title="选择目标表">
    进入用于保存提交结果的表格。
  </Step>
  <Step title="新建表单视图">
    打开视图侧边栏，点击 `+`，选择 **表单视图**。
  </Step>
  <Step title="设置名称和协作方式">
    输入表单视图名称，并设置为协作视图或个人视图。
  </Step>
  <Step title="创建视图">
    点击创建后，进入表单构建器继续配置字段。
  </Step>
</Steps>

## 配置表单字段

表单字段会根据表格中的字段生成。你可以在表单构建器中调整字段显示。

| 操作 | 说明 |
| --- | --- |
| 调整顺序 | 拖拽字段，改变表单中的显示顺序 |
| 隐藏字段 | 点击字段右上角的隐藏按钮，或把字段拖到左侧隐藏区域 |
| 添加字段 | 从左侧字段列表拖入表单，或点击 `+ 添加字段到此表` |
| 修改显示名称 | 点击字段后修改表单中显示给填写者的名称 |
| 添加说明 | 为字段添加副标题，补充填写说明 |
| 设置必填 | 开启必填后，填写者必须完成该字段才能提交 |

如果把勾选字段设为必填，填写者必须勾选该字段后才能提交表单。

## 表单外观

部分计划支持为表单添加封面图片或标志。封面区域会按固定高度展示，重要文字或关键信息不适合放在容易被裁剪的图片中。

## 分享表单

表单配置完成后，点击视图栏中的 **分享表单**，复制表单链接并发送给填写者。也可以点击预览，在浏览器中查看实际提交页面。

## 提交设置

| 设置 | 说明 |
| --- | --- |
| 设置密码 | 填写者需要输入正确密码后才能提交 |
| 查看提交表单的人 | 要求填写者登录 Teable，并把提交者信息记录到创建者字段 |

开启“查看提交表单的人”后，未登录的填写者需要登录或创建 Teable 账户后才能提交。

## 条件逻辑和复杂提交页

表单视图适合轻量收集，支持字段显示名、副标题、必填和基础分享设置。如果需要条件逻辑、多步骤流程、品牌化页面或更复杂的交互，可以使用 [Teable AI App Builder](/zh/basic/ai/app-builder) 创建提交应用。

常见做法：

- 用一句话描述页面，先生成基础版本。
- 从当前数据库创建应用，让应用和数据保持关联。
- 在应用构建器中继续调整流程、样式和字段交互。
- 发布应用后，将公开访问链接分享给填写者。

如果是在数据库内部做临时数据分析或提问，请使用 [AI Chat](/zh/basic/ai/ai-chat)，而不是应用构建器。

更多 AI 工作流请看 [AI 概览](/zh/basic/ai/overview)。
