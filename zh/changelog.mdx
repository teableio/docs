# AI 上下文、短链与性能优化

## 功能更新

* **新增 Teable Chat App 上下文能力**：Teable Chat 可将已授权且被 @提及的 App 作为上下文，根据其代码、结构和配置提供更准确的 AI 协助，同时自动排除敏感的运行时配置。

* **新增分享与模版短链功能**：支持使用更简洁的链接进行分享，便于传播和使用。

## 问题修复&优化

* **提升回收站稳定性**：修复打开包含批量删除记录的回收站时可能出现的错误，并提升大批量记录的加载与恢复可靠性。

* **提升跨 Base 计算字段更新稳定性**：解决计算字段更新可能相互阻塞或无法完成的问题，确保数据及时、准确地更新。

* **优化视图切换性能**：减少重复加载和不必要的刷新，让视图切换更流畅。

* **优化字段计算状态展示**：提供更清晰的计算进度与状态变化，简化活动面板并补充更多翻译，帮助用户快速了解字段计算情况。

* **优化大批量删除的回收站预览**：支持展示部分已删除记录及超出数量，帮助用户快速了解删除内容。

* **提升大型表格加载性能**：优化多列表格的加载响应速度，让复杂表格的浏览与操作更加流畅。

- **修复手动排序异常**：确保实时更新或重新加载页面后，记录顺序仍与手动排序结果保持一致。

- **优化自链接记录选择器**：仅展示当前连接中可见的字段，并返回完整记录内容，让记录选择和后续使用更加准确。

- **优化表格描述布局**：缩小表格描述的最大宽度，避免遮挡视图标签页，使表头布局更清晰、操作更方便。

- **统一链接字段可见性规则**：通过 API 调整可见字段时，链接字段及链接分享配置会始终保留被链接表的主字段，与界面操作保持一致。

- **提升附件与数据导入稳定性**：增强受信任来源下附件访问和数据导入请求的可靠性，减少运行时配置环境中的失败情况。

- **修复分享 Base 另存为副本失败**：当插件面板超出分享范围时，仍可正常创建副本；副本仅包含已分享的表格和面板，并会安全跳过无效内容。

- **修复视图持续显示“计算中”**：改进公式与 Lookup 字段的计算恢复机制，减少视图长时间卡在计算状态的问题。

- **优化公式型 Lookup 字段体验**：确保此类字段可正常编辑和转换，并提升其加载、保存、显示设置及错误处理的可靠性。

</Update>

# 性能、搜索与稳定性优化

## 问题修复&优化

* **提升大规模数据操作性能**：优化大量汇总字段复制、内联表创建和批量删除等场景，减少高容量工作流的处理开销，操作更流畅、更高效。

* **优化汇总字段更新效率**：改进汇总数据的回填和按需读取能力，加快计算字段更新，同时确保结果准确。

* **优化批量删除流程**：减少删除生命周期中的重复处理，同时保持批量大小控制、撤销与重做、实时更新、审计日志、自动化及回收站等现有能力不变。

* **修复引用字段删除异常**：解决删除仍被其他功能引用的字段时可能发生崩溃的问题；删除确认对话框现在可以稳定打开，确认和取消操作也更加可靠。

* **修复生成应用首次 OAuth 登录失败**：使用域登录或开放登录的新用户现在可以正常创建账号，不再出现“Failed to create user record”错误，提升首次登录成功率。

* **修复展开记录中的隐藏字段显示问题**：拥有权限的用户启用“显示隐藏字段”后，可以在展开记录中正确查看隐藏字段值；系统仅在需要时加载相关内容，减少不必要的请求。

* **保留隐藏字段显示偏好**：更新后将继续沿用用户已有的“显示隐藏字段”设置，包括旧版本保存的偏好，避免重复配置。

* **优化共享视图中的隐藏字段加载**：隐藏字段将遵循共享表的访问范围，避免展开其他表的链接记录时出现加载失败。

* **修复新建 Base 的访问状态记录**：新创建的 Base 现在会被正确标记为已访问，确保后续流程能够准确识别并提供一致体验。

* **改善嵌入配置体验**：增加页面底部间距，并修复嵌入设置弹出层可能被遮挡或无法操作的问题，让配置内容更易查看和调整。

* **修正 OAuth 授权范围**：访问令牌现在仅包含用户明确批准的权限，避免应用获得未授权的额外访问能力。

* **优化大型工作区完整性检查**：减少链接字段元数据的重复扫描，提升完整性检查和修复任务的执行效率与可靠性，降低大型工作区的数据库负载。

* **优化大型数据集的表格搜索性能**：提升高流量场景下的搜索速度，并增强搜索结果的可靠性，让大型表格中的查找体验更稳定。

* **提升自动化邮件处理稳定性**：优化邮箱连接异常后的自动恢复能力，并避免处理已失效的邮件结果，减少自动化任务中断。

# 计算字段与稳定性优化

## 功能更新

* 为计算字段（包括 formulas、lookups 和 rollups）新增可见的计算活动状态，让用户更清楚地了解表格值何时仍在计算中。

## 问题修复&优化

* **优化搜索高亮下的表格选择效果**：开启全局搜索高亮时，选中的行或单元格更容易与搜索结果区分。

* **优化窄列中的统计信息显示**：当列宽较窄时，会优先保留百分比等关键数值，减少统计结果被截断导致的误解。

* **修复分组视图中新建记录的分组字段问题**：即使分组字段被隐藏，从某个分组中新建记录时，也会自动带入正确的分组值。

* **优化关联记录后的计算结果更新**：在更新用户信息后立即创建关联记录时，lookup、formula 等计算字段能更快得到正确结果，减少短时间为空或不完整的情况。

* **修复仅修改字段名称或描述时可能保存失败的问题**：只编辑字段名称或描述时，不会误触发计算字段回填，字段数据和计算配置会保持稳定。

* **提升 rollup 等计算字段配置的编辑稳定性**：减少编辑相关字段配置时可能出现的异常。

* **修复嵌套 lookup 字段重命名后的配置丢失问题**：重命名嵌套 lookup 字段时，会保留原有的 single-select 选项配置，已显示的记录值也会保持正常。

* **优化 Cookie 处理稳定性**：提升了部分部署环境下 Cookie 的兼容性与稳定性，减少相关功能异常。

</Update>

# 复制与计算性能优化

## 问题修复&优化

* **保留外部链接字段数据**：复制包含外部链接字段的表时，支持保留已关联的记录值，包括部分单向外部链接和有序链接内容。

* **提升表复制性能**：复制包含自链接表或外部链接字段的表时速度更快，减少大型数据复制时的等待时间。

* **优化复制 Base 的体验**：复制较大的 Base 时处理更高效，进度响应更及时。

* **增强自链接记录复制稳定性**：复制自链接记录时，关联关系保留得更可靠，减少链接丢失或异常的情况。

* **增加复制性能覆盖**：补充了外部链接字段相关的复制性能保障，降低后续版本出现性能回退的风险。

* **修复公开分享表单提交异常**：优化公开分享表单的提交处理，减少因路由判断异常导致的 500 错误；匿名提交、需登录分享和隐藏字段等原有行为保持不变。

* **优化条件 Rollup 性能**：在使用字段引用进行条件判断时，`count`、`sum`、`average` 等 Rollup 计算更快，尤其适用于大量关联记录的场景。

* **提升复杂 Rollup 计算稳定性**：优化包含筛选、分组记录、`array_join` 排序结果、限制数量、Lookup 排序等组合场景的计算性能。

* **加快无关联字段表的复制速度**：复制不包含 Link 字段的表时速度更快，并能更稳定地保留记录 ID 和已存储的计算字段值。

* **保障关联字段表复制结果正确**：包含 Link、Lookup、Rollup 或跨表/跨 Base 引用的表，复制后相关引用关系会保持隔离与正确。

* **修复 FIND / SEARCH 公式计算异常**：当公式对多选、关联记录等 JSON 类型字段使用 `FIND` 或 `SEARCH` 时，计算结果更加稳定。

* **优化大规模公式重算可靠性**：提升大量文本搜索类公式级联重算时的稳定性。
