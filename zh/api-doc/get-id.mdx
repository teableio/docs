---
title: "获取 ID"
---

## 空间 ID (SpaceId)

点击目标空间，复制 URL 上以 spc 开头的字符串，示例: `spcXXXXXXXXXX`

![](/images/2026-02-12-10-43-08.png)

## 数据库 ID (BaseId)

点击目标数据库，复制 URL 上以 bse 开头的字符串，示例：`bseXXXXXXXXXX`

![](/images/2026-02-12-10-46-49.png)

## 表格 ID (TableId)

点击目标表格，复制 URL 上以 tbl 开头的字符串, 示例：`tblXXXXXXXXXX`

![](/images/2026-02-12-10-48-18.png)

## 视图 ID (ViewId)

点击目标视图，复制 URL 上以 viw 头的字符串, 示例：`viwXXXXXXXXXX`

![](/images/2026-02-12-10-48-11.png)

## 字段 ID (FieldId)

**方法 1：**

点击表格左上角 `更多` 中的 `设计` 按钮，进入设计界面，查看所有字段的 ID

<img src="/images/2026-02-12-11-18-41.png" width="400" />

![](/images/2025-02-14-18-16-26.png)

**方法 2：**

双击列标题或选择“编辑字段”后打开`高级属性`。在`物理字段名`部分，您可以找到字段 ID。

<img src="/images/2026-02-12-11-19-48.png" width="400" />

<img src="/images/2026-02-12-11-20-06.png" width="400" />

## 记录 ID (RecordId)

展开记录编辑卡片，复制 URL 上 recordId= 后方以 rec 开头的字符串, 示例：`recXXXXXXXXXX`

![](/images/2026-02-12-10-55-17.png)
