## 在 Teable 中集成 AI 模型

### 步骤：

1. **打开一个空间**：首先，在 Teable 中导航并进入您希望集成 AI 模型的特定空间。
2. **进入设置界面**：在当前空间的界面中，找到并点击右上角的“设置”选项。
3. **进入AI配置**：在设置菜单中，选择并点击“AI配置”。
4. **添加 AI 集成**：在 AI配置 界面，点击“+ 添加模型提供商”按钮。
5. **配置模型信息：**
   - 为您的模型提供一个名称。
   - 选择模型提供商。
   - 设置您的模型提供商的基础 URL。
   - 输入相应的 API Key。
6. **完成添加**：配置完成后，点击“添加”按钮。

![](/images/2025-09-25-21-21-06.png)

完成以上步骤后，您将能够在 Teable 的 AI 功能中看到“空间模型”和“系统模型”的区分。您刚刚配置的 AI 模型将显示在“空间”栏目下。

![](/images/2025-06-09-10-57-08.png)

<Tip>除了设置空间级别的AI API，Saas 版本的企业版或者私有化的版本可以在管理员面板里设置组织级别的AI API。</Tip>
