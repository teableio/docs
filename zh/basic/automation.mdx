AI 驱动 Teable 的每一环节，将您的数据在几秒钟内转化为智能、全自动化的工作流。从智能字段处理到端到端自动化，AI 处理每一步，让您专注于最重要的工作。

Teable 自动化让任何人都能构建强大的工作流，无需编写代码。只需用自然语言告诉 AI 您的需求即可。

## 从零开始用 AI 创建自动化

最简单的方式。打开任意表格，点击右侧边栏的 **AI 对话**，描述你想要的效果：

- *"新订单进来时，发一条 Slack 通知，包含客户名和金额。"*
- *"每周一早上，给我发一封包含所有逾期任务的邮件。"*
- *"有人提交反馈表单时，自动分类并更新分类字段。"*

AI 为你创建整个自动化 — 触发器、操作和字段映射。在自动化面板中检查并启用。

## 从触发器开始用 AI 构建自动化

你也可以从一个具体的触发器开始，让 AI 来构建操作：

1. **添加触发器** — 选择触发类型（如"记录创建时"）并配置。
2. **添加操作** — 点击 **+**，进入 **用 AI 创建**，选择 **运行脚本**。
3. 在配置面板中，点击 **打开编辑器**。
4. 从 **建议列表** 中快速选择，或用自然语言描述需求。
5. AI 生成 JavaScript 代码，检查后点击 **应用**。
6. 点击 **测试** 用真实数据运行，然后 **启用** 工作流。

## 手动构建自动化

你也可以使用内置操作逐步搭建：

1. **添加触发器** — 选择触发类型，选择表，配置字段。点击 **测试**。
2. **添加操作** — 选择操作类型（创建记录、发送邮件、HTTP 请求等），通过 **+** 插入动态变量。点击 **生成预览** 或 **按配置执行**。
3. **启用** — 点击左上角开关。

## 编辑运行中的工作流

修改会保存为草稿，线上工作流继续使用旧版本，直到你点击 **应用更新**。

## 执行历史

打开工作流 → **执行历史**。每次运行显示状态、耗时和逐步输入/输出。点击可查看详情。

## 限制

| 项目 | 限制 |
|---|---|
| 邮件发送速率 | 5 封 / 秒 / Base |
| Webhook 接收速率 | 50 次 / 秒 / Base，2 次 / 秒 / 工作流 |
| Webhook 请求体大小 | 4 MB |
| 运行配额与历史保留 | 按套餐不同 |
