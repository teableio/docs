---
title: "Kanban"
description: "Display records as cards grouped by a field, useful for tracking status, stages, and workflows."
---

Kanban view displays records as cards and groups them into columns by field value. Use it for task status, sales stages, content workflows, recruiting pipelines, and other stage-based work.

## Use Cases

| Scenario | Good for |
| --- | --- |
| Project tasks | Track tasks by Not started, In progress, Done |
| Sales pipeline | Manage customers by Lead, Opportunity, Negotiation, Closed |
| Content workflow | Track content by Idea, Writing, Review, Published |
| Hiring pipeline | Manage candidates by Screening, Interview, Offer, Hired |

## Create a Kanban View

<Steps>
  <Step title="Create a view">
    Click `+` at the top of the view sidebar.
  </Step>
  <Step title="Choose Kanban view">
    Select **Kanban view** from the view types.
  </Step>
  <Step title="Choose a grouping field">
    Choose the field used to create columns, such as status, stage, owner, or date.
  </Step>
</Steps>

## Group Settings

Kanban view supports most field types except attachment fields as the grouping field.

| Action | Description |
| --- | --- |
| Change grouping field | Click the dropdown icon next to a group title and choose another grouping field |
| Hide empty groups | Turn on **Hide empty groups** in group settings |
| Collapse groups | Choose **Collapse group** from the group menu to show the group as a vertical bar |
| Sort groups | When grouped by a Single Select field, drag groups to reorder them |

## Manage Groups

When the grouping field is a Single Select field, you can manage options directly from the board:

- Click **Add group** to create a new option.
- Choose **Rename group** from the group menu to rename an option.
- Choose **Delete group** from the group menu to delete an option.

These actions change the options in the corresponding Single Select field. Confirm the impact before changing groups used by other views or workflows.

## Configure Cards

Click **Card configuration** in the toolbar to control what appears on each card.

| Setting | Description |
| --- | --- |
| Visible fields | Choose which fields appear on cards |
| Field order | Adjust the order of fields on cards |
| Cover | Choose an attachment field as the card cover |
| Hide field names | Show only field values to reduce card density |

Show only fields needed to understand the record status, such as title, owner, due date, priority, and key tags.

## Move and Edit Cards

- Drag a card to change its order or move it to another group.
- Click `+` at the bottom of a group to create a record in that group.
- Right-click a card to insert a card above or below, expand the card, or delete it.
- Click a card, or choose **Expand card**, to edit the record in detail.

## Notes

- When you drag a card to another group, Teable updates the value of the grouping field.
- For team workflows, consider locking important views so grouping and card settings are not changed by accident.
