---
title: "画册"
description: "用画册视图以图片卡片展示记录，适合产品目录、素材库和作品集。"
---

画册视图会把记录展示为图片卡片。它适合以图片为主要内容的表格，比如产品图片、设计稿、活动素材、照片库或作品集。

## 适用场景

| 场景 | 适合内容 |
| --- | --- |
| 产品目录 | 商品图片、型号、价格、状态 |
| 素材管理 | 设计稿、活动素材、广告图、截图 |
| 作品集 | 项目封面、案例图、设计作品 |
| 图片审核 | 按状态、负责人或标签筛选图片记录 |

## 创建画册视图

<Steps>
  <Step title="新建视图">
    点击视图侧边栏顶部的 `+`。
  </Step>
  <Step title="选择画册视图">
    在视图类型中选择 **画册视图**。
  </Step>
  <Step title="配置卡片">
    选择封面图片字段，并按需配置筛选、排序和卡片字段。
  </Step>
</Steps>

## 设置封面图片

画册视图通常使用附件字段作为封面。

| 设置 | 说明 |
| --- | --- |
| 封面字段 | 选择要作为卡片封面的附件字段 |
| 多张图片 | 一条记录有多张图片时，默认使用第一张作为封面 |
| 适应显示 | 控制图片是否适应卡片区域 |

## 配置卡片信息

点击工具栏中的 **卡片设置**，可以选择卡片上显示哪些字段，也可以调整字段顺序和是否显示字段名称。

建议只保留识别记录所需的关键字段，比如名称、状态、负责人、价格或标签。字段过多会降低画册视图的浏览效率。

## 查看和编辑图片

- 点击图片，可以查看大图。
- 在大图模式下，可以切换图片、缩放查看细节或下载原图。
- 点击卡片信息区域，可以打开记录并编辑字段内容。

## 管理记录

- 点击底部的 **新建**，可以添加记录并上传图片。
- 右键点击卡片，可以删除记录或执行其他记录操作。
- 配合筛选和排序，可以按状态、分类或负责人管理图片记录。

## 注意事项

- 画册视图适合以附件图片为核心的表格。如果记录主要是文本或数字，表格视图通常更高效。
- 画册视图中的字段显示配置只影响当前视图，不会删除字段内容。
