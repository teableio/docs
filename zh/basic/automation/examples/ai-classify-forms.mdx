<Tip>我们推荐直接在 AI 对话中构建自动化示例。只需描述你想要的工作流，AI 会为你完成全部设置，包括触发器、操作、脚本和配置。</Tip>

## 用 AI 创建

打开表格右侧边栏的 AI 对话，告诉 AI 你想要什么。例如：

*"有人提交反馈表单时，将反馈分类为 Bug 报告、功能需求或一般反馈，并写回记录"*

AI 会自动创建完整的工作流。你可以查看生成的脚本，用真实数据测试，确认无误后启用。

## 手动设置

1. **创建工作流**，触发器选择 **表单提交时**。
2. 选择你的**反馈**表和对应表单。
3. 点击**测试**。
4. **添加操作** → **AI 生成**。
   - 提示词：
     ```
     Classify this feedback into exactly one category:
     Bug Report, Feature Request, or General Feedback.

     Reply with only the category name.

     Feedback: <通过 + 插入反馈字段>
     ```
   - 输出类型：**文本**
5. **添加操作** → **更新记录**。
   - 记录 ID：触发记录的 ID（通过 **+** 插入）
   - 分类字段 → AI 生成的输出（通过 **+** 插入）
6. 逐步测试每个步骤，然后**启用**。

## 提示

- 如需更精细的分类，可以在提示词中增加更多类别。
- 使用 **JSON** 输出类型，可以让 AI 在一次调用中返回多个字段（如分类 + 情感倾向 + 摘要）。
