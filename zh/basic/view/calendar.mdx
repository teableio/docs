---
title: "日历"
description: "用日历视图按日期查看和管理记录，适合会议、活动、排期和项目时间线。"
---

日历视图会把记录显示在日历中。只要记录中有日期字段，就可以按日期查看任务、会议、活动、发布计划或项目节点。

## 适用场景

| 场景 | 适合内容 |
| --- | --- |
| 会议和预约 | 会议时间、客户拜访、面试安排 |
| 项目排期 | 任务开始日期、截止日期、里程碑 |
| 内容计划 | 发布日期、活动日期、营销排期 |
| 跨天事项 | 有开始日期和结束日期的项目、活动或请假记录 |

## 创建日历视图

<Steps>
  <Step title="新建视图">
    点击视图侧边栏顶部的 `+`。
  </Step>
  <Step title="选择日历视图">
    在视图类型中选择 **日历视图**。
  </Step>
  <Step title="选择日期字段">
    选择用作开始时间的日期字段。需要展示跨天事项时，也可以选择结束日期字段。
  </Step>
</Steps>

## 日期字段设置

| 设置 | 说明 |
| --- | --- |
| 开始日期字段 | 决定记录显示在哪一天 |
| 结束日期字段 | 用于展示跨天或有持续时间的记录 |

如果没有合适的日期字段，请先在表格中创建日期字段。

## 颜色设置

日历记录可以使用颜色区分不同类型。

| 颜色方式 | 说明 |
| --- | --- |
| 自定义颜色 | 为当前日历视图使用固定颜色 |
| 根据字段 | 根据字段值显示不同颜色，适合按状态、类型或负责人区分记录 |

## 浏览日历

- 使用左右箭头切换月份。
- 点击 **今天** 回到当前日期。
- 点击日历中的记录，可以打开并编辑记录详情。

## 创建和调整事件

- 点击日期格子，可以创建一条新记录。
- 拖动选择日期范围，可以创建跨天记录。
- 拖拽已有记录到其他日期，可以调整日期字段。
- 如果设置了结束日期字段，可以通过拖拽调整持续时间。

## 注意事项

- 日历视图依赖日期字段。删除或更改日期字段后，请检查日历视图配置。
- 日历视图只改变记录的展示方式，不会改变记录本身的字段结构。
