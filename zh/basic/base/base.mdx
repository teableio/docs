---
title: "数据库"
---

数据库是组织和管理相关表格的容器。每个空间可包含多个数据库，每个数据库可包含多个表格。而每个数据库都可以独立工作，为特定项目或表格集合提供清晰的信息结构。

对于不熟悉数据库的用户来说，可以将数据库类比为Excel中的工作簿，每个工作簿可以有多张表。

### 添加数据库

1. 进入空间
2. 点击右上角`创建数据库`

![](/images/ji-ben-gong-neng/4994637b0704b22f.png)

### 重命名数据库

1. 进入空间
2. 将鼠标悬停在目标数据库上
3. 点击`···`按钮打开菜单
4. 点击`重命名`

### 删除数据库

1. 进入空间
2. 鼠标悬浮在目标数据库上
3. 点击`···`按钮打开菜单
4. 点击`删除`

  src="/images/ji-ben-gong-neng/8e0f0b62392c4d11.png"

### 复制数据库到另一个空间

第一步：在当前空间中选择要复制的数据库

第二步：点击数据库菜单图标并选择`复制数据库`

第三步：在弹出的窗口中选择目标空间，需拥有目标空间的`可搭建`权限才能完成复制

<img
  src="/images/ji-ben-gong-neng/8acd3a13937871cc.png"
  style={{width: "300px", height: "auto"}}
  className="mx-auto"
/>

### 导入数据库

1. 进入空间
2. 点击空间右上角`···`按钮打开菜单
3. 点击`导入`
4. 上传 .tea 文件确认导入

![](/images/2025-07-28-12-29-49.png)

### 导出数据库

1. 进入空间
2. 鼠标悬浮在需要导出的数据库上
3. 点击`···`按钮打开菜单
4. 点击`导出`
5. 导出完成后，将通过通知栏提醒，可直接在通知中下载 .tea 文件

![](/images/2025-07-28-12-32-23.png)


### 数据库内的功能

数据库可包含多个表格，用于组织和管理工作相关的各类信息。

例如，客户管理数据库可包含"客户公司"、"客户联系人"、"客户跟进记录"等表格；会议室管理数据库则可包含"会议室信息"、"设备管理"、"预约记录"等表格。

因此，数据库的主要功能都围绕表格展开

- **创建表格**：可根据项目需求创建新表格，自定义表格结构、字段类型和数据格式

- **编辑表格**：支持对现有表格进行编辑，包括添加、删除和修改记录

- **导出和导入数据**：支持将数据导出为 CSV 格式，也可导入 CSV 和 XLSX 格式的数据文件

详情请参阅[表格相关章节](/zh/basic/table)了解更多信息.

### 进阶

在 Teable 中创建数据库相当于在 PostgreSQL 中创建 Schema，开发者可利用此特性获得强大的 SQL 数据访问能力和数据库集成功能。详情请参阅[通过 SQL 访问表格数据](/zh/api-doc/sql-query)章节。
