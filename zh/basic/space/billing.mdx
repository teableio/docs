## 访问账单设置

只有**空间所有者**才能访问账单设置。访问方式：

1. 点击空间右上角的「···」按钮
2. 选择**空间设置**
3. 导航到**计费**页面

![](/images/2025-12-10-23-03-03.png)

## 计费页面概览

计费页面展示以下信息：

### 当前订阅计划

显示当前订阅状态，包括：

- **订阅级别**（免费版、高级版、专业版、团队版或企业版）
- **席位单价**（月付或年付）
- **席位数量**
- **续费日期**或取消日期（如适用）

点击**改变订阅计划**按钮可以升级、降级或修改订阅。

### 用量统计

通过以下指标监控空间的资源消耗：

| 指标 | 描述 |
|------|------|
| **协作者总数** | 当前空间及其数据库的协作者数量 |
| **记录总数** | 空间内所有表格的总行数 |
| **附件容量** | 上传附件使用的总存储空间 |
| **邮件总数** | 通过自动化使用 Teable 邮件服务发送的邮件数量 |

<Tip>空间回收站和数据库回收站中的数据也会计入记录总数统计。如需减少记录总数，请清理「空间 → 回收站」或「数据库 → 回收站」中的数据。</Tip>

### 算力

算力用于 AI 功能和自动化：
- 自动化操作
- AI 字段生成
- AI 对话
- 应用构建器

算力部分显示您的总算力额度和当前使用量。点击**详情**可查看按功能和月份的算力消耗明细。

### 结算明细

显示您的账单信息：
- **姓名**：账单账户名称
- **邮箱**：账单邮箱地址

点击**管理信息**可打开支付门户，在其中可以：
- 更新支付方式
- 修改账单地址
- 查看支付历史

<Note>「管理信息」按钮会在新标签页中打开 Stripe 客户门户。</Note>

### 发票

查看和下载账单发票：

1. 滚动到页面底部的**发票**部分
2. 在表格中找到您需要的发票
3. 点击下载图标获取 PDF 副本

发票表格显示以下信息：
- **参考号**：发票编号
- **含税总额**：含税金额
- **状态**：支付状态（如已支付）
- **日期**：发票日期
- **下载**：PDF 下载按钮

## 更改订阅计划

要升级、降级或修改订阅计划：

1. 进入**空间设置** → **计费**
2. 点击**改变订阅计划**按钮
3. 您将被重定向到**订阅计划**页面
4. 选择您想要的订阅级别
5. 选择计费周期（月付或年付）
6. 如需调整席位数量
7. 完成结账流程

<Tip>年付比月付节省约 17%。</Tip>

### 可用计划

| 计划 | 适合 |
|------|------|
| **免费版** | 个人用户 |
| **高级版** | 小型团队 |
| **专业版** | 成长型企业 |
| **企业版** | 大型团队和公司 |

## 取消订阅

要取消订阅：

1. 进入**空间设置** → **计费**
2. 点击**改变订阅计划**进入订阅计划页面
3. 找到当前计划并点击取消选项
4. 确认取消

<Warning>
取消后：
- 您的订阅将在当前计费周期结束前保持有效
- 您会看到「将于 [日期] 取消」的提示
- 您的数据将被保留，但付费功能在周期结束后将不可用
- 您可以随时重新订阅以恢复付费功能的访问权限
</Warning>

## 常见问题

<AccordionGroup>
  <Accordion title="谁可以访问账单设置？">
    只有空间所有者可以访问账单设置。其他协作者（可搭建、可编辑、可评论、可查看）无法查看或修改账单信息。
  </Accordion>
  
  <Accordion title="席位是如何计算的？">
    席位根据您空间及其数据库中的协作者总数计算。邀请更多协作者会自动增加您的席位数量和账单金额。
  </Accordion>
  
  <Accordion title="什么时候会扣费？">
    订阅后会立即扣费，之后在每个计费周期（月付或年付）的续费日期扣费。
  </Accordion>
  
  <Accordion title="如何更新支付方式？">
    点击结算明细部分的**管理信息**按钮。这会打开 Stripe 客户门户，您可以在其中更新银行卡或添加其他支付方式。
  </Accordion>
  
  <Accordion title="可以退款吗？">
    如需退款，请联系我们的支持团队 support@teable.ai。退款政策可能因您的订阅类型和使用情况而异。
  </Accordion>
  
  <Accordion title="支付失败会怎样？">
    如果支付失败，我们会通过邮件通知您。您将有一个宽限期来更新支付方式，超过宽限期后订阅将被暂停。
  </Accordion>
  
  <Accordion title="算力用于什么？">
    算力被 AI 功能消耗，包括自动化 AI 操作、AI 字段生成、AI 对话和应用构建器。每个计划都包含每月的算力配额。
  </Accordion>
</AccordionGroup>
