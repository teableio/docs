路径：管理面板 → AI 设置

### 1. AI 能力设置

- 说明：
  - 实例管理员可以独立控制每一项 AI 功能是否在实例内可用。
  - 列表中的功能包括：
    - AI 基础能力：包括 AI 表格查询工具、执行 SQL 查询工具、生成脚本工具、表格数据可视化工具。
    - 创建数据库：从 0 开始创建整个数据库。
    - 创建应用：基于数据库创建应用。
    - 创建自动化：从 0 开始创建整个自动化。
    - 表格资源操作：通过 AI 对表格资源操作，可以创建、修改、删除表格、视图、字段、记录。
    - 建议：根据当前数据库数据以及对话内容，给予建议。

### 2. 自定义 AI 模型

- 用于激活或停用当前实例内的自定义 AI API 配置。

![](/images/2025-10-10-21-04-09.png)

- 说明：
  - 关闭时：AI 功能将被禁用或使用系统默认值（如果可用）。
  - 打开时：将使用配置的 AI 模型，开启前，请配置好高级对话模型。

> [自定义模型配置指南](/zh/basic/ai/custom-model)

### 3. 模型提供商

- 用于管理自定义配置 AI 模型提供商的接入。

### 4. 模型偏好

- 为不同类型的任务场景指定默认使用的对话模型，优化成本和性能。
  - 高级对话模型：
    - 用于 AI Chat 主对话、复杂推理和代码生成任务。
    - 推荐：`claude-sonnet-4.5` 或具有强推理能力的模型。
  - 中型对话模型：
    - 用于 AI 公式生成。
    - 未配置时会回退到高级对话模型。
    - 推荐：`claude-haiku-4.5`
  - 小型对话模型：
    - 用于自动生成对话标题和数据库名称等轻量级任务。
    - 未配置时会回退到高级对话模型。
    - 推荐：`claude-haiku-4.5`

![](/images/2025-10-10-21-04-33.png)

### 5. 高级对话模型能力

- 测试所选"高级对话模型"的多模态处理能力。
- 说明：
  - 可以通过"测试模型"按钮来验证所选的高级模型是否支持处理图片和 PDF 内容。

### 6. 嵌入模型

- 可选。用于文档问答 - 当用户上传文档并询问文档内容相关问题时使用。

### 7. 应用构建器 (App Builder)

配置相关设置以启用 [应用构建器](/zh/basic/ai/app-builder) 功能，从您的数据生成自定义 Web 应用程序。

**可用配置项：**

- **v0 API 密钥** (必需)：从 [v0.app/chat/settings/keys](https://v0.app/chat/settings/keys) 获取，需要付费 v0 订阅
- **自定义域名**：为发布的应用配置您自己的域名
- **Vercel API Token**：用于自定义域名绑定
- **API 代理 URL**：用于 API 访问受限的地区
- **AI Gateway API Key**：用于集中监控和成本控制

> 详细配置说明请参阅 [应用构建器私有化部署配置](/zh/basic/ai/app-builder#私有化部署配置)

### 8. 网络搜索

- 配置 Firecrawl API 密钥以启用网络搜索功能，访问 Firecrawl 设置 获取 API 密钥
