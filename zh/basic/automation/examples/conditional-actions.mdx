---
title: "根据记录值执行不同操作"
description: "使用条件逻辑为工作流添加分支，根据字段值执行不同的操作"
---

**目标：** 当创建支持工单时，根据优先级将其发送到不同的邮箱——紧急工单发给值班团队，其他工单发给通用队列。

## 步骤

1. **创建工作流**，触发器选择 **记录创建时**。
2. 选择你的**工单**表。
3. 点击**测试**获取示例记录。
4. **添加条件**节点。
5. 设置规则：**优先级** `是` **紧急**。
6. 点击**测试**。
7. **在 true 路径上** → 添加**发送邮件**操作：
   - 收件人：`oncall@yourcompany.com`
   - 主题：`紧急工单：` + 工单标题（通过 **+** 插入）
   - 正文：工单描述（通过 **+** 插入）
8. **在 false 路径上** → 添加**发送邮件**操作：
   - 收件人：`support@yourcompany.com`
   - 主题：`新工单：` + 工单标题
   - 正文：工单描述
9. 分别测试两条路径，然后**启用**。

## 提示

- 你可以嵌套条件：在 false 路径中再添加一个条件，将"高"优先级和"低"优先级分开处理。
- 使用 AND/OR 组合规则——如 `优先级 是 紧急 AND 负责人 为空`。
